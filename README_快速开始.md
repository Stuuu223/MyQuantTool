## 🚀 快速启动

### 方式1：使用启动脚本（推荐）
双击 `启动系统.bat` 即可一键启动系统。

### 方式2：手动启动
```bash
# 激活虚拟环境（如果存在）
venv\Scripts\activate

# 启动系统
streamlit run main.py
```

系统将在浏览器中自动打开 http://localhost:8501

## 🛠️ 故障排查

### UI显示异常
- 检查浏览器控制台错误（F12）
- 刷新页面
- 重启系统

### 数据加载失败
- 检查网络连接
- 检查AkShare接口状态
- 等待几秒后重试

### 性能问题
- 清理 `__pycache__` 文件夹
- 重启系统
- 减少扫描股票数量

### Git冲突
- 参考 `GIT_PULL_HANDLE_PROMPT.md`
- 参考 `GIT冲突解决记录_20260117.md`
