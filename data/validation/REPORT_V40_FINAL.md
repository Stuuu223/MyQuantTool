## 三、技术实现细节

### 3.1 手术一：日内动能净值

```python
# 废除：today_change_pct > 5.0
# 改用：price_momentum = (Close - Low) / (High - Low)
price_momentum = (today_close - today_low) / (today_high - today_low)
if (price_momentum >= 0.90 and volume_ratio >= 2.0) or volume_ratio > 3.0 or is_yesterday_limit_up:
    passed.append(stock)
```

**物理意义**: 不看涨了多少，只看收盘价是否死死咬住最高点，反映资金强顶最高点的决绝度

### 3.2 手术二：L1探针增量换手

```python
# 废除：turnover_rate > 0.5 (全天累计)
# 改用：delta_turnover = (current_volume - old_volume) / float_volume
if delta_turnover > 0.5 and price_change_pct < 0.5:
    return False  # 派发信号
```

**物理意义**: 5分钟增量换手才是真实区间做功，累计换手是偷懒

### 3.3 手术四：动能衰竭止盈法

```python
# 废除：max_profit_pct > 0.15 或 0.08 固定百分比
# 改用：
# 法则1：天量大阴线
if volume > (yesterday_vol * 2.0) and close < (open * 0.97):
    exit_reason = '天量大阴线(主力崩塌)'

# 法则2：双重断头台
if close < ma5 and close < vwap:
    exit_reason = '动能枯竭(双破MA5与VWAP)'
```

**物理意义**: 不看赚了多少，只看主力是否撤退
