### 2.3 Phase 3: 分析分数天花板问题

**根因定位**:

```python
# kinetic_core_engine.py 原代码

# 动能分：当inflow_ratio >= 1.0%时直接撞天花板！
kinetic_score = min(30.0, (inflow_ratio_pct / 1.0) * 30.0)  # 满分30

# 势能分：当ratio_stock >= 15时直接撞天花板！  
potential_score = min(30.0, (ratio_stock / 15.0) * 30.0)  # 满分30

# 价格推力：涨停时day_strength=1.0，满分40
momentum_score = day_strength * 40.0  # 满分40

# 时间衰减：current_time = 15:00（尾盘）
if current_time.hour >= 14:
    multiplier *= 0.5  # 尾盘腰斩！
```

**计算链条**:
- base_score = 40 + 30 + 30 = 100
- multiplier = 1.5 (sustain>1.2) × 1.2 (inflow>1.5%) × 0.5 (尾盘) = **0.9**
- final_score = 100 × 0.9 = **90分**

**这就是全员90分的原因！**

## 三、解决方案

### 3.1 手术1: 时间冻结修复

**位置**: `main.py` L485

```python
# 修复前
frozen_time = dt_class.combine(dt_class.today(), time_type(15, 0, 0))

# 修复后
frozen_time = dt_class.combine(dt_class.today(), time_type(9, 45, 0))
```

**物理意义**: 09:45是morning_confirm区间，时间衰减=1.0（无衰减），对齐实盘引擎的盘后模式。

### 3.2 手术2: 恢复90th分位防线

**位置**: `main.py` L463-475

```python
# 【CTO V50核心】90th分位动态量比防线！
if volume_ratios and len(volume_ratios) > 0:
    import numpy as np
    vr_values = [v for v in volume_ratios.values() if v > 0]
    if vr_values:
        vr_90th = np.percentile(vr_values, 90)
        min_vr_threshold = max(vr_90th, 1.5)  # 物理下限1.5x
        # 过滤：只保留量比>=90th分位的尖刀！
        base_pool = [s for s in base_pool if volume_ratios.get(s, 0) >= min_vr_threshold]
```

**效果**: 1370只 → **156只尖刀**

### 3.3 手术3: 废除[:300]硬编码

**位置**: `logic/data_providers/universe_builder.py` L126-148

```python
# 修复前
final_pool = sorted_step3[:300]  # 硬编码截断！

# 修复后
final_pool = sorted(step3, key=lambda x: self._volume_ratios.get(x, 0), reverse=True)
# 不再截断，让90th分位防线在scan_cmd中执行压缩
```

### 3.4 手术4: 砸碎kinetic_score天花板

**位置**: `logic/strategies/kinetic_core_engine.py` L408-420

```python
# 修复前：inflow>=1%就撞天花板
kinetic_score = min(30.0, (inflow_ratio_pct / 1.0) * 30.0)

# 修复后：让INFLOW%差异体现！
# 7.43%流入 → 7.43*5 = 37.15分
# 3.62%流入 → 3.62*5 = 18.1分
# 差距拉开！
kinetic_score = min(inflow_ratio_pct * 5.0, 40.0)
```

**改进建议**:
1. 增加止盈机制：最大收益>15%时触发跟踪止损
2. 增加止损机制：回撤>8%时强制离场
3. 缩短持仓周期：从榜单出现到离场不超过3天
