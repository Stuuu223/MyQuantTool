# 连板首板起爆特征研究报告 V3.0

**版本**: V3.0 (CTO审计修复版)  
**生成时间**: 2026-03-06  
**样本量**: 2527个连板事件（≥3连板）  
**数据窗口**: 2024-01-01 ~ 2026-03-05

## 执行摘要

本研究通过连板序列检测器（V3）分析A股2527个连板事件的起爆特征，发现：

1. **真正建仓窗口**：T-5到T-1（换手率从2.9%爬升至3.7%）
2. **首板日特征**：换手率中位4.96%，是背景的1.7倍
3. **暗流信号**：换手率从<3%异动至>3.5%但价格未动

**完整性检查**：T-1涨停率0.0%，T0涨停率100.0%，校验通过。

---

## 一、研究背景与目标

### 1.1 问题起源

- **时间**：2026年2月~3月
- **触发事件**：V1研究报告结论与实战经验不符
- **核心困惑**：V1报告显示T-1涨停比例86.3%，意味着"起爆前一天已是涨停"，逻辑矛盾

### 1.2 研究目标

本研究目标：从历史数据反推"连板首板"的量化特征，构建可落地的选股规则。

**核心问题**：
1. 真正的首板日（T0），换手率在什么区间？
2. 首板前是否有"缩量整理"或"暗流异动"？
3. 如何避免"样本污染"导致的错误结论？

### 1.3 研究假设

- H1：真正的首板日，换手率应处于"温和放量"区间
- H2：首板前存在"低换手整理期"或"暗流异动期"
- H3：V1报告的样本存在"连板中间截片污染"

**关键说明**：
- 涨停判断：主板10%（9.5-10.5%），注册制20%（19-21%），ST股5%（4.5-5.5%）
- 流通股本使用QMT实时值，对2024-2026数据影响可控

### 2.2 核心算法

#### 2.2.1 连板序列检测器（V3核心创新）

废弃V1的滑动窗口扫描，改用连板序列检测器：

```
算法：
1. 逐日扫描是否涨停
2. 连续涨停日归为同一个 series
3. 中断即结算，记录 series_start（第1板）
4. 只保留 ≥3连板的 series

保证：同一股票不会有重叠或嵌套的 series
```

#### 2.2.2 换手率计算

```
换手率 = 成交量（手） × 100 / 流通股本（股） × 100%
```

#### 2.2.3 K线形态分类（V3修复）

```python
if is_limit_up and abs(body_pct) < 0.3:
    candle_type = "一字涨停"  # V1误判为"平盘"
elif is_limit_up and body_pct >= 0.3:
    candle_type = "涨停"
elif body_pct > 5:
    candle_type = "大阳"
# ...
```

### 4.2 核心发现

**发现1：暗流期换手异动**
- T-5到T-1：换手率从2.94%爬升至3.69%
- 涨幅中位仅0.12%~0.62%，价格几乎未动
- 这是"量在价先"的典型特征

**发现2：首板日特征**
- T0换手中位：4.96%（是背景2.56%的1.94倍）
- T0涨停比例：100%
- 一字涨停占比：约50%

**发现3：连板期间换手递增**
- T0：4.96%
- T+1：4.88%
- T+2：7.04%
- T+3：11.39%（分歧加大）

## 五、策略建议

### 5.1 实战筛选逻辑

```python
def is_limit_up_ignition_candidate(stock, today):
    """连板首板候选股筛选"""
    
    # 条件1：当日换手在异动区间
    if not (3.0 <= today_turnover <= 6.0):
        return False
    
    # 条件2：换手率相比背景放大>1.5倍
    if turnover / bg_turnover_20d < 1.5:
        return False
    
    # 条件3：前5日换手在暗流区间（2.5-4%）
    if not (2.5 <= avg_turnover_t5 <= 4.0):
        return False
    
    # 条件4：当日涨幅>0（右侧确认）
    if today_return <= 0:
        return False
    
    # 条件5：价格在MA20上方（趋势确认）
    if close < ma20:
        return False
    
    return True
```

### 5.2 分档策略

| 连板目标 | 换手特征 | 买入时机 | 止损条件 |

### 5.3 风险警示

```python
def is_death_pattern(stock, today):
    """死亡模式识别"""
    
    # 死亡模式1：高换手首板（换手>15%）
```

### 6.1 当前研究局限

#### 已知偏差
1. **幸存者偏差**：股票池为当前A股，已退市股票未纳入
2. **浮动分母问题**：FloatVolume使用当前值，未考虑历史增发/回购
3. **一字板无法买入**：T0约50%是一字涨停，实战无法买入
