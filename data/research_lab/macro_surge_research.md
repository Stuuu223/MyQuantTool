# 暴力真龙日K宏观物理光谱扫描

## 研究背景

由于QMT基础接口对分K/Tick数据有近一个月时间限制，173样本库中包含2025年的历史数据无法获取分K。为突破这一限制，我们降维到日K级别进行宏观物理特征提取。

## 研究方法

**样本来源**：`data/validation/recent_samples_2025_2026.csv`中label=1的暴力真龙样本

**分析工具**：`tools/macro_surge_scanner.py`

**有效样本**：125只

## 核心物理量定义

### 1. 宏观做功推力 (Macro MFE)

```
Macro MFE = 振幅% / 成交额(亿)
```

**物理意义**：每亿成交额推高了多少振幅百分比。
- 值越低，说明资金做功效率越高（用更少的钱推更高的幅度）
- 封板样本MFE=0.36，未封板样本MFE=1.00

### 2. 宏观纯度 (Macro Purity)

```
Macro Purity = (Close - Low) / (High - Low) × 100%
```

**物理意义**：收盘价在日内振幅中的位置。
- 100% = 收盘在最高点（涨停）
- 0% = 收盘在最低点
- 封板样本纯度100%，未封板样本纯度39%

### 3. 起爆放量倍数 (Vacuum Ratio)

```
Vacuum Ratio = 当日成交额 / 前5日成交额均值
```

**物理意义**：起爆日相对于历史均值的放量程度。
- 中位数1.68x，说明真龙起爆日成交额是历史均值的1.68倍
- 过高（>3x）可能是对倒诱多，过低（<1x）可能是缩量洗盘

### 4. 起爆前量比

```
Pre Volume Ratio = 前5日成交量均值 / 前20日成交量均值
```

**物理意义**：起爆前的缩量/放量状态。
- 中位数1.74x，说明真龙起爆前已有资金潜伏迹象

## 扫描结果

### 整体统计

| 指标 | 中位数 | 物理意义 |
|------|--------|----------|
| Macro MFE | 0.52 | 做功效率较高 |
| 宏观纯度 | 100% | 大部分收盘封板 |
| 起爆放量倍数 | 1.68x | 温和放量 |
| 起爆前量比 | 1.74x | 有资金潜伏 |
| 封板率 | 80.8% | 大部分封板 |

### 封板 vs 未封板对比

| 指标 | 封板样本(101只) | 未封板样本(24只) | 差异 |
|------|----------------|-----------------|------|
| Macro MFE | **0.36** | 1.00 | 封板效率高2.8倍 |
| 宏观纯度 | **100%** | 39.2% | 封板纯度高60.8pp |
| 放量倍数 | 1.73x | 1.56x | 差异不大 |

**关键发现**：
> 封板样本的Macro MFE显著低于未封板样本(0.36 vs 1.00)，
> 说明封板真龙的资金做功效率更高——用更少的钱推更高的幅度。

## 实战应用

### 真龙识别标准

基于宏观物理光谱，真龙应具备：
1. Macro MFE < 0.5（资金做功效率高）
2. 宏观纯度 > 80%（收盘接近高位）
3. 放量倍数 1.5-3.0x（温和放量，非对倒）
4. 起爆前量比 > 1.5x（有资金潜伏）

### 假龙排雷

以下特征需警惕：
1. Macro MFE > 1.0（资金做功效率低）
2. 宏观纯度 < 40%（收盘在水下）
3. 放量倍数 > 5x（过度放量，疑似对倒）

## 结论

日K宏观物理光谱扫描揭示了暴力真龙的核心物理特征：
- **高效率做功**：Macro MFE低，用更少钱推更高幅度
- **高位收盘**：纯度接近100%，收盘在最高点附近
- **温和放量**：放量倍数1.5-2x，非极端对倒
- **资金潜伏**：起爆前量比>1.5x，有主力建仓痕迹

这些宏观特征可用于快速筛选潜在真龙，与微观分K特征形成互补。
