# 界面说明

## 右上角菜单说明

由于 Streamlit 是国际化的开源框架，右上角的系统菜单（Deploy、Settings）默认显示为英文，这是框架本身的设计，无法直接修改为中文。

### 📋 菜单功能说明

#### ⚙️ Settings（设置）
- **功能**：调整应用显示设置
- **选项**：
  - Theme（主题）：Light（浅色）/ Dark（深色）
  - Color（颜色）：选择界面配色
  - Font Size（字体大小）：调整文字大小
- **建议**：推荐使用 Light 主题，更清晰易读

#### 🚀 Deploy（部署）
- **功能**：将应用部署到云端
- **说明**：
  - 需要注册 Streamlit Community Cloud 账号
  - 部署后可以通过网址访问
  - 免费版有运行时间限制
- **建议**：个人使用无需部署，本地运行即可

#### ❌ Clear cache（清除缓存）
- **功能**：清除应用缓存数据
- **说明**：
  - 清除已缓存的历史数据
  - 重置应用状态
  - 刷新后重新加载数据
- **建议**：遇到数据异常时使用

#### 📤 Run on Cloud（在云端运行）
- **功能**：在 Streamlit Cloud 上运行
- **说明**：与 Deploy 功能类似
- **建议**：个人使用无需此功能

#### 📖 About（关于）
- **功能**：查看应用信息
- **说明**：显示版本号和系统信息
- **建议**：一般不需要使用

---

## 💡 使用建议

### 日常使用
1. **本地运行**：使用 `streamlit run main.py` 或 `start.bat`
2. **主题选择**：Settings → Theme → Light（推荐）
3. **字体大小**：Settings → Font Size → Medium（推荐）

### 遇到问题
1. **数据异常**：点击 Clear cache 清除缓存
2. **显示异常**：刷新浏览器页面
3. **性能问题**：减少查看的数据量或时间范围


---

## ⚠️ 注意事项

1. **Deploy 功能**：需要网络连接和账号，个人使用不需要
2. **Clear cache**：会清除所有缓存数据，需要重新加载
3. **Settings**：设置保存在浏览器本地，清除浏览器数据会丢失

## 📞 常见问题

### Q1：为什么右上角菜单是英文？
**A**：Streamlit 是国际化框架，系统菜单默认为英文，这是框架设计，无法修改。

### Q2：需要使用 Deploy 功能吗？
**A**：不需要。个人使用本地运行即可，Deploy 是用于将应用发布到云端供他人访问。

### Q3：如何更改界面主题？
**A**：点击右上角 ⚙️ Settings → 选择 Theme → Light 或 Dark

### Q4：Clear cache 会删除我的数据吗？
**A**：不会删除您的配置文件和自选股，只会清除临时缓存的数据。

### Q5：如何恢复默认设置？
**A**：在侧边栏的"🔧 配置管理"中点击"重置配置"按钮。

---

**记住**：右上角的英文菜单是 Streamlit 框架的系统功能，不影响您使用本工具的核心功能。所有重要的功能都在界面中已经中文化了！
