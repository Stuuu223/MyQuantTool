## 二、正确做法

### 应该自动扫描"高潮窗口"
不是人工指定09:35，而是：

```python
def find_climax_window(resampled_df, min_change_pct=3.0):
    """
    自动寻找真正的起爆高潮窗口
    
    策略1: 资金量最大窗口
    策略2: 涨幅最大窗口  
    策略3: 量价共振窗口 (放量+涨幅)
    """
    # 计算每个窗口的价格变化
    resampled_df['price_change_pct'] = resampled_df['lastPrice'].pct_change() * 100
    
    # 计算每个窗口的资金强度得分
    resampled_df['intensity_score'] = (
        resampled_df['vol_delta'] * resampled_df['price_change_pct']
    )
    
    # 找出强度得分最高的窗口
    climax_idx = resampled_df['intensity_score'].idxmax()
    climax_row = resampled_df.loc[climax_idx]
    
    return climax_idx, climax_row
```

## 四、发散性思考：资金维度做到极致

既然老板说"资金为王"，我们应该：

### 1. 多层资金切片
不只是5分钟，还要：
- 1分钟超短脉冲
- 5分钟起爆确认
- 15分钟趋势延续
- 60分钟主升浪

### 2. 资金流向纯度
不只看过成交额，要看：
- **主动净买入** = 向上跳动成交 - 向下跳动成交
- **大单占比** = 大单成交额 / 总成交额
- **资金集中度** = Top10分钟资金 / 全天资金

### 3. 跨日资金记忆
不只盯单日，要连起来看：
- 12.31收盘资金强度 → 隔日预期
- 01.05开盘资金承接 → 接力确认
- 两日资金差值 → 加速信号
