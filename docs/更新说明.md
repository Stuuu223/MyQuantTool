## 🚀 使用指南

### 首次使用
1. 确保 Python 环境已安装
2. 安装依赖：`pip install -r requirements.txt`
3. 访问 https://siliconflow.cn/ 获取免费 API Key（2000万 tokens）
4. 在侧边栏配置管理中填入 API Key
5. 点击"保存配置"

### 单股分析
1. 输入 6 位股票代码（如 600519）
2. 选择开始日期
3. 查看技术指标和 K 线图
4. 点击"呼叫 AI 投顾"获取专业建议

### 多股对比
1. 切换到"多股对比"标签页
2. 输入要对比的股票代码（用逗号分隔）
3. 点击"开始对比分析"
4. 查看对比表格和收益率曲线

### 策略回测
1. 切换到"策略回测"标签页
2. 输入回测股票代码和日期范围
3. 调整回测参数（ATR倍数、网格比例、手续费）
4. 点击"运行回测"
5. 查看回测报告和交易记录

## 📊 技术指标说明

### MACD（异同移动平均线）
- **MACD > 信号线**：多头市场，趋势向上
- **MACD < 信号线**：空头市场，趋势向下
- **柱状图**：显示多头/空头力量强弱

### RSI（相对强弱指标）
- **RSI > 70**：超买，可能回调
- **RSI < 30**：超卖，可能反弹
- **30-70**：正常区间

### 布林带
- **上轨**：价格阻力位
- **中轨**：价格中枢
- **下轨**：价格支撑位
- **接近上轨**：注意风险
- **接近下轨**：可能反弹
