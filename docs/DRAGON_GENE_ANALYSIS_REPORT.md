## 执行摘要

本报告通过深度分析四只股票的Tick级历史数据，揭示了**真龙**与**假龙**的关键基因差异。研究发现，真龙在**形态位置**、**竞价态度**、**封板结构**三个维度上具有显著区别于假龙的特征。

**核心发现**:
- 真龙空间差 < 10%，假龙空间差 > 15%
- 真龙竞价高开 +2%以上，假龙竞价低开或平开
- 真龙早板（10:00前）涨停，假龙下午勉强上板
- 真龙封板坚决，假龙反复开板

## 六、真龙筛选算法改进方案

### 6.1 当前算法局限

现有算法主要关注:

**缺失维度**:

### 6.2 V2.0改进方案

#### 6.2.1 新增形态位置过滤器

```python
def filter_breakthrough_purity(stock_data):
    """
    突破纯度过滤器
    排除空间差>15%的票（套牢盘太多）
    """
    space_gap_pct = calculate_space_gap(stock_data)
    
    if space_gap_pct > 15:
        return False, f"空间差{space_gap_pct}% > 15%，套牢盘过多"
    
    if space_gap_pct < 5:
        return True, f"空间差{space_gap_pct}% < 5%，高突破纯度"
    
    return True, f"空间差{space_gap_pct}%，可接受"
```

#### 6.2.2 新增竞价态度评分

```python
def score_auction_attitude(stock_data):
    """
    竞价态度评分
    高开幅度反映资金坚决度
    """
    gap_pct = stock_data['auction_gap_pct']
    
    if gap_pct >= 10:
        return 100, "一字开盘，极度强势"
    elif gap_pct >= 5:
        return 80, "大幅高开，资金抢筹"
    elif gap_pct >= 2:
        return 60, "适度高开，态度积极"
    elif gap_pct > 0:
        return 40, "小幅高开，态度一般"
    else:
        return 0, "低开或平开，资金犹豫"
```

#### 6.2.3 新增封板质量评分

```python
def score_limit_up_quality(stock_data):
    """
    封板质量评分
    上板时间越早、开板次数越少越好
    """
    time_to_limit = stock_data['time_to_limit']
    open_count = stock_data['open_count']
    
    # 上板时间评分 (越早越好)
    if time_to_limit < "09:35:00":
        time_score = 100
    elif time_to_limit < "10:00:00":
        time_score = 80
    elif time_to_limit < "10:30:00":
        time_score = 60
    elif time_to_limit < "11:30:00":
        time_score = 40
    else:
        time_score = 20
    
    # 开板次数扣分 (越少越好)
    open_penalty = min(open_count * 2, 40)  # 最多扣40分
    
    final_score = time_score - open_penalty
    return max(final_score, 0)
```

#### 6.2.4 综合评分公式

```python
def calculate_dragon_score(stock_data):
    """
    真龙综合评分算法 V2.0
    """
    # 基础分 (0-100)
    base_score = stock_data['base_score']
    
    # 形态位置权重 30%
    purity_weight = 0.30
    purity_score = 100 if stock_data['space_gap_pct'] < 10 else 50
    
    # 竞价态度权重 30%
    auction_weight = 0.30
    auction_score, _ = score_auction_attitude(stock_data)
    
    # 封板质量权重 25%
    limit_weight = 0.25
    limit_score = score_limit_up_quality(stock_data)
    
    # 资金纯度权重 15%
    capital_weight = 0.15
    capital_score = stock_data['capital_purity_score']
    
    # 综合评分
    final_score = (
        base_score * 0.2 +  # 基础分权重降低
        purity_score * purity_weight +
        auction_score * auction_weight +
        limit_score * limit_weight +
        capital_score * capital_weight
    )
    
    return final_score
```

## 八、结论与建议

### 8.1 核心结论

1. **真龙与假龙的差异是系统性的**，不是偶然的
2. **形态位置（空间差）是最重要的区分维度**，决定套牢盘多少
