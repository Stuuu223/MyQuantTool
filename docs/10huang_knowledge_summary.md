## 一、选股三板斧核心逻辑

### 核心文章
- http://www.10huang.cn/xuangu/53343.html - 选股三板斧教程：前言
- http://www.10huang.cn/xuangu/53346.html - 选股三板斧教程3：如何判断主力

### 三板斧内容

**第一板斧：判断风口**
- 市场主流热点、题材方向
- 板块共振判断（Leaders≥3, Breadth≥35%）

**第二板斧：判断股性**
- 股票活跃度、涨停历史
- 经常涨停，且涨停后第二天大都有溢价
- 很少坑人的
- 涨停后溢价率高

**第三板斧：判断主力**
- 主力资金成本、介入程度
- **主力是推动龙头股的力量**，没有主力就没有行情
- **跟随主力成本越近**：风险越低，收益越大
- **偏离主力成本越远**：风险越大，收益越低

### 核心逻辑
```
股价 = 基本面 + 择时
触发大涨前提 = 符合市场风口 + 符合龙头股股性 + 接近主力成本价买入
```

## 二、竞价战法操作要点

### 核心文章
- http://www.10huang.cn/zhangting/32432.html - 集合竞价选涨停的操作
- http://www.10huang.cn/zhangting/51885.html - 首板竞价技巧
- http://www.10huang.cn/fenshitu/51246.html - 竞价预期使用

### 操作条件
- 股票波段涨幅小于25%
- 开盘一小时股价处于强势中
- 回调不破开盘价、均价线或前日收盘价

### 竞价技巧
1. **有重大题材或个股利好**：可采用竞价排单买入
2. **以涨停价格买入**：找到可能是龙头的个股
3. **竞价以一字板涨停大量**：成功率高

### 竞价预期规则
```
规则一：明牌焦点股，竞价不能超预期 → 打折扣
规则二：明牌股，竞价超预期 → 不要轻易卖
```

### 量比排序使用
- **9:25竞价出来后**，9:30开盘后15分钟内完成
- **量比排序** + **涨幅排序**配合使用
- **有效价格范围内**选取成交量最大的价位

### 竞价强弱判断
- 观察一字涨停数量及封单
- 比较两日竞价的一字板数量
- 数量越多 → 资金流入意愿强

## 三、半路战法时间特征

### 核心文章
- http://www.10huang.cn/zhangting/29723_all.html - 股票半路初段战法
- http://www.10huang.cn/zhangting/46509_all.html - 龙头战法之三种核心半路反包模式
- http://www.10huang.cn/zhangting/35938.html - 半路板超短技术

### 半路定义
- **股价在上涨过程中选择跟进**
- 必须是股价有上涨的苗头了才能跟
- 不是在低开低走的时候跟进

### 最佳买点特征
- **分时上从犹豫到果断的那个点** = 半路最好的买点
- **也是最确定的买点**

### 平台型反包半路模式
1. 首先有一波拉升，拉升过程中至少要有两个涨停板高度
2. 第一波拉升后，股价进入平台调整
3. **平台时间最少3天，最多不超过10天**

### 分时特征
- 开盘后长时间围绕均线不动
- 上下震荡1个点之内
- **震荡时间越长**，启动拉出涨停概率越大
- 震荡过程中下方严重缩量

## 四、低吸战法买入时机

### 核心文章
- http://www.10huang.cn/buy/46586.html - 几种低吸买入方法
- http://www.10huang.cn/zhangting/53263.html - 首板低吸的两个黄金买点
- http://www.10huang.cn/buy/42741.html - 涨停板低吸战法

### 低吸买入时机
- 股价连续三个交易日不破首板开盘价
- **三个交易日后（第四天）择机买入**
- 股价或股指一路沿着20日均线反压振荡下行
- 在低位企稳时，以一根带量中阳冲过20日均线

### 涨停板低吸战法
- **趋势股涨停后跌3-5个点** → 一般是震仓
- 这个时候选择买入
- 第二天或第三天重新启动走强概率大

### 龙头股低吸逻辑分类
```
潜伏式低吸：
  - 逻辑尚未发酵，买进去等待逻辑走出来
  - 风险：预判错了逻辑没有发酵、小方向错误
```

### 买入位置
- 尽量远离前期高点或前期筹码密集套牢区的压力
- 或者刚好突破前期高点的调整机会
- 回调高度距离高点一般20点左右
- 回调到5日线或10日线的缩量急杀处低吸
- 调整周期一般在2-5个交易日内

## 五、龙头战法识别标准

### 核心文章
- http://www.10huang.cn/xuangu/53777.html - 龙头战法的关键
- http://www.10huang.cn/zhangting/47628_all.html - 龙头战法赚钱效应七个要点
- http://www.10huang.cn/zhangting/54240.html - 深度剖析市场龙头

### 龙头战法核心
- **判断主流板块是关键**，占60%成功因素
- 必须自己经历几轮板块轮动，才能切身判断

### 龙头识别标准
- **市场中最具影响力**的个股
- **涨幅最大**
- **最受资金关注**
- 需要涨到**2-3板及以上**才能相对确定龙头地位

### 赚钱效应识别标准
- 大盘无大跌风险
- 核心板块上涨突出
- 空间领头羊持续创新高

### 选股逻辑顺序
```
1. 等待板块三日后
2. 筛选出连续涨停的个股
3. 加以研究与跟踪
4. 市场已经淘汰不纯粹的个股
```

### 一进二模式要点
- 图表形态漂亮，突破向上或前期无大的套牢盘
- **竞价爆量跳空高开**（竞价最后一刻是红柱）
- 结合板块、情绪冰点、有市场地位预期差更好

## 六、选股逻辑与筛选标准

### 核心文章

- http://www.10huang.cn/zhangting/50283_all.html - 1进2首板接力模式

### 强势股筛选顺序
```
1. 选择强势板块
2. 涨幅榜及涨停分析
3. 快速涨速榜
4. 最高涨幅榜
5. 振幅+涨幅榜
```

### 1进2选股标准
- 选择是首板票（昨日首次涨停）
- 只有强势股才有资金关注
- 只有强势才能涨停
- 必须做强势股

### 首板抓热点概念方法
- 直接找高开概念，且此概念在上升趋势中
- 不看下跌趋势
- 重点看高开概念中的新概念（三四天刚形成，或最近1个月左右刚形成）
- 快速找出横盘

### 长逻辑波段票筛选
- 优先选择长逻辑还没破的
- 加分项：近端有还没被兑现的催化
- 情绪炒作而没有吸引中线资金逻辑的票，情绪退潮后就散了

## 八、卖点判断标准

### 核心文章
- http://www.10huang.cn/fenshitu/45323.html - 分时图的绝佳卖点：量比指标卖出法
- http://www.10huang.cn/macd/30267_all.html - 5个MACD最佳卖点
- http://www.10huang.cn/Kxian/53244.html - 最最常见的几个见顶K线

### 量比指标卖出法
- **股价与量比指标相背离**：股价走高，量比指标走平或走低 → 卖出信号
- **股价与量比指标同向走低**：股价从高位走低，量比指标同步走低 → 卖出信号
- **一顶比一顶低**：出现三个以上的顶峰，且顶峰一个比一个低 → 最佳卖点

### MACD卖点
- **零轴下方DIFF线死叉DEA线**：死叉日卖出（强烈卖出信号）
- **价格、MACD背离卖出**：股价虚浪拉升创出新高，MACD不能同步创出新高 → 见顶信号

### BOLL上轨卖点
- 当股价突破BOLL上轨时 → 看跌卖出信号

### 高位放量卖点
- 高位时不能见到巨量
- 一旦见到巨量说明是短期高点

### 见顶K线形态
- 射击之星、吊颈线、穿头破脚K线、穿头破脚组合和乌云盖顶
- 出现在连涨后高位 → 卖出信号

## 九、买点判断标准

### 核心文章
- http://www.10huang.cn/Kxian/49323.html - 12种绝佳买入形态K线图
- http://www.10huang.cn/buy/55114.html - 股价突破前期高点是较佳买入时机
- http://www.10huang.cn/fenshitu/44068.html - 分时图拉升前的买点

### 股价突破前期高点买点
- 股价有效突破了以前的高点 → 套牢盘变成获利盘
- 股价会继续上涨，直至再遇到更高的高点

### 股价突破历史天价买点
- 股票放量突破历史天价 → 重要买入时机
- 后市一般会有较大的上升空间
- 一般会有题材配合
- 逆市上涨创新高的股票更值得关注

### 分时图三次脉冲放量拉升买点
- 成交量三次脉冲放大 → 主力动用大笔资金拉升股价
- 买入信号
- 成交量越大，股价短线大涨概率越高

### 强力试盘后的买点
- **突破买点**：突破上影线高点时买入（确认度高）
- **低吸买点**：需要对分时盘口的强度进行确认

### 周KDJ买点
- 周KDJ中的KD两线底背弛后金叉 → 绝对的买入信号

### 多指标配合法则
- **定义**：每一个买卖点出现，都需要至少两个或两个以上的技术分析工具发出同方向买卖信号
- **示例**：筹码分布指标与MACD指标配合
- 目的：提高买卖点准确性

### ROC动量指标
- 建议使用**双重时间框架**
- 长周期动量指标判断方向
- 短周期动量指标寻找买卖点

### WR威廉指标
- 选择股价异动前的瞬间
- 能选择出加速下跌的瞬间
- 能选择出涨升启动前的最佳入市时机

### EMV指标
- 涵盖价格与成交量两个因素
- 强调单位成交量的股价变动
- 衡量单位成交量的股价变动

### 技术分析五要素
```
开盘价、收盘价、最高价、最低价、成交量
```

## 十一、风险控制策略

### 仓位管理
- 大盘处在高位（如五千点），长期潜伏失败几率大
- 超短期潜伏在大盘崩溃前预期兑现，能博得确定性收益

### 止损策略
- 选择错误，及时止损
- 预判大方向对了，但小方向错误 → 及时止损

### 情绪周期控制
- 情绪下降期休息
- 不要强行操作

### 基本面风险
- 商誉减值黑天鹅
- 高商誉、财报亏损等需要避开
- 使用通达信"扫雷避险"功能
