# WORKLOG 2026-03-13

## 今日主题：近三年暴力样本换手率深度研究

---

## 一、研究背景

### 1.1 问题起源
V129-V131四维共振漏斗上线后，发现5日均换手率>5%的阈值可能过于严格，导致"冷启动暴拉"真龙被误杀。

### 1.2 研究目标
- 分析近三年暴力样本的换手率特征
- 验证5%阈值的合理性
- 建立完整的暴力样本数据库

---

## 二、研究方法

### 2.1 阶梯式样本筛选
```
2天涨幅 > 15%
3天涨幅 > 25%
4天涨幅 > 35%
5天涨幅 > 60%
6天涨幅 > 100%
10天涨幅 > 70%
```

### 2.2 数据采集
- 时间范围：2023-03-14 ~ 2026-03-13（近3年）
- 数据源：QMT本地日K数据
- 样本量：**63,963个暴力事件**
- 涉及股票：4,609只

### 2.3 换手率计算
```python
# 流通股本反推法（避免TrueDictionary缓存问题）
est_market_cap = avg_amount_60d / 0.025  # 假设日均换手2.5%
float_volume = est_market_cap / avg_price * 100

# 换手率计算
turnover_pct = (volume * 100 / float_volume) * 100
```

## 三、核心发现

### 3.1 样本分布

| 阶梯 | 样本数 | 涨幅中位 |
|------|--------|---------|
| 2天>15% | 33,235 | 21.0% |
| 3天>25% | 15,395 | 33.1% |
| 4天>35% | 8,830 | 46.3% |
| 5天>60% | 2,127 | 61.1% |
| 6天>100% | 217 | 131.7% |
| 10天>70% | 4,159 | 82.3% |

### 3.2 换手率画像（关键发现）

#### 2天暴力型（33,235个样本）
```
起爆前换手率：T-7(2.87%) → T-1(6.00%) → T0(8.39%)
5日均换手(不含T0): 4.73%
被5%阈值过滤：52.6%
```

#### 6天暴力型（217个样本，最极端）
```
起爆前换手率：T-7(0.18%) → T-1(6.48%) → T0(8.84%)
5日均换手(不含T0): 3.59%
被5%阈值过滤：64.3%
```

#### 10天暴力型（4,159个样本）
```
起爆前换手率：T-7(5.87%) → T-1(12.01%) → T0(13.94%)
5日均换手(不含T0): 10.28%
被5%阈值过滤：21.2%
```

### 3.3 核心结论

| 类型 | 5日均换手中位 | <5%占比 | 特征 |
|------|--------------|---------|------|
| 短期暴力型(2-6天) | 3.5-6% | 44-64% | 冷启动暴拉 |
| 长期暴力型(10天) | 10.28% | 21.2% | 持续放量 |

**结论**：短期暴力爆发型是"冷启动暴拉"，换手率从极低位突然爆发；长期暴力型是"持续放量型"，起爆前换手率已经很高。
