## 九、CTO V154 核心反应堆拆解（2026-03-14）

### 🔬 第一站：MFE做功效率算子

**代码位置**：`kinetic_core_engine.py:300-310`

```python
upward_thrust = ((price - low) + (high - open_price)) / 2
price_range_pct = upward_thrust / prev_close * 100.0
mfe = price_range_pct / inflow_ratio_pct
efficiency_multiplier = 3.0 / (1.0 + math.exp(-2.0 * (mfe - 1.2)))
```

**物理意义**：
- `upward_thrust`：上推力向量，主力向上做功的距离
- `mfe`：每单位流入创造的涨幅 = 做功效率
- 高MFE = 少量资金大涨幅 = 真空板
- 低MFE = 大量资金小涨幅 = 烂板

**判别效果**：
| 类型 | 涨幅 | 流入 | MFE | 判决 |
|------|------|------|-----|------|
| 烂板 | 8% | 5% | 1.6 | 乘数≈1.5 |
| 真空板 | 8% | 0.5% | **16.0** | 乘数≈3.0 |
