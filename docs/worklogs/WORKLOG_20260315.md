**文件**: `tools/mock_live_runner.py:191`

est_price = current_tick.get('price', 10) if tick_list else 10

est_price = float(tick_list[-1].get('price', 10)) if tick_list else 10.0

**Git**: `c215237`

### 修复5: 循环内return导致榜单清空

**文件**: `tools/mock_live_runner.py:383`

**问题描述**:

**修复方案**:
```python

# 修复后（只跳过当前股票）:

```

### 修复12: Law1 抗重力姿态判定

**文件**: `logic/strategies/kinetic_core_engine.py`

**问题描述**:
- 原逻辑: `drawdown_from_high = (high - price) / prev_close > 3.5%`
- 问题: 用昨收做分母，低开股票误判，高开股票漏判
- 案例: 昨收10元，今开9.5元拉到10.5元，洗盘回10.15元，回撤3.5%被误杀

**修复方案**:
```python
# Law1: 抗重力姿态
price_momentum = (price - low) / (high - low)
if price_momentum < 0.57:
    return 0.0  # 姿态破败
```

**物理意义**:
- 现价在日内价格区间的相对位置（无量纲）
- 1.0 = 最高点（完美姿态）
- 0.57 = 从高点回落43%振幅（临界点）
- 0.0 = 最低点（完全失守）

**Git**: `c9365c1`

### 修复13: Law2 真实做功效率

**文件**: `logic/strategies/kinetic_core_engine.py`
