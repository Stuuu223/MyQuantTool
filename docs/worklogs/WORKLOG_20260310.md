### 9.3 修复内容

#### 修复#1：1m净流入近似算法（K线动力学）
```python
# CTO级 1m 净流入近似算法 (K线动力学)
# 核心逻辑：如果收盘>开盘，视该分钟成交额为正向做功；反之为负向损耗
def calculate_net_flow_from_1m(df_slice):
    net_flow = 0.0
    for _, k in df_slice.iterrows():
        k_range = k['high'] - k['low']
        if k_range > 0:
            power_ratio = (k['close'] - k['open']) / k_range
            net_flow += k['amount'] * power_ratio
        else:
            if k['close'] >= k['open']:
                net_flow += k['amount'] * 0.5  # 一字板半量计入
    return max(0, net_flow)
```

#### 修复#2：MFE数学保护

```python
# 废除归零逻辑，用最小摩擦力底线保护
MIN_FRICTION_RATIO = 0.05  # 全市场最低自然摩擦力

safe_inflow_ratio = max(inflow_ratio, MIN_FRICTION_RATIO)
mfe = amplitude_pct / safe_inflow_ratio

```

- 投研版Token仅能补充近期数据，无法获取2年历史

### 10.3 数据飞轮策略调整
1. **主力**：MiniQMT日常订阅累积数据
2. **补漏**：投研版补充近期缺失数据（1年内）
3. **回测**：需要长历史时用日K或分K聚合
