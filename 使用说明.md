## 系统简介

MyQuantTool 是一个功能强大的 A 股智能量化终端，集成了多种技术分析、战法识别、风险管理和智能推荐功能，帮助投资者提升交易效率和决策质量。

### 核心特性

- 📊 **全面的技术分析**：支持单股分析、多股对比、策略回测
- 🎯 **智能战法识别**：集合竞价、龙头战法、热点题材挖掘
- ⚠️ **完善的风险管理**：仓位管理、止损止盈、风险预警
- 🤖 **智能推荐系统**：每日报告、战法推荐、市场分析
- 📈 **丰富的可视化**：多种图表类型，直观展示数据
- 📱 **移动端支持**：响应式设计，支持手机访问

## 快速开始

### 系统要求

- Python 3.8+
- 网络连接（获取实时数据）
- 推荐使用 Chrome 或 Edge 浏览器

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://gitee.com/stu223/MyQuantTool.git
   cd MyQuantTool
   ```

2. **创建虚拟环境**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   source .venv/bin/activate  # Linux/Mac
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **启动应用**
   ```bash
   streamlit run main.py
   ```

   或使用启动脚本：
   ```bash
   start.bat  # Windows
   ```

## 日常使用流程

### 推荐的每日使用流程

#### 第一步：查看每日报告 🤖

1. 打开 **🤖 智能推荐** 标签页
2. 选择 **每日报告** 功能
3. 点击 **📊 生成今日报告** 按钮

**报告内容：**
- 市场情绪分析
- 涨跌情况统计
- 操作建议
- 推荐战法

#### 第二步：检查风险预警 ⚠️

1. 打开 **⚠️ 风险管理** 标签页
2. 选择 **止损止盈检查** 功能
3. 输入持仓股票的代码、当前价格、买入价格
4. 点击 **📊 检查** 按钮

**检查内容：**
- 是否触发止损
- 是否达到止盈
- 当前盈亏比例
- 操作建议

#### 第三步：分析自选股 📊

1. 在左侧控制台选择要分析的股票
2. 打开 **📊 单股分析** 标签页
3. 查看技术指标和趋势分析
4. 查看资金流向和龙虎榜数据

#### 第四步：查看热点题材 🎯

1. 打开 **🎯 热点题材** 标签页
2. 点击 **🔍 开始扫描** 按钮
3. 查看热点板块和龙头股
4. 分析题材持续度

#### 第五步：保存分析记录 📜

1. 在分析完成后，系统会自动保存记录
2. 打开 **📜 历史记录** 标签页
3. 可以查询、导出历史分析记录

- 分析题材持续度（上升期、活跃期、衰退期、震荡期）

#### 仓位计算

**功能：** 根据风险比例和止损比例计算建议仓位

**使用方法：**
1. 输入总资金
2. 设置单笔风险比例（1-10%）
3. 设置止损比例（2-10%）
4. 点击 **📊 计算仓位** 按钮

**输出结果：**
- 建议仓位金额
- 仓位占比
- 单笔最大损失

#### 止损止盈检查

**功能：** 检查持仓是否触发止损或止盈

**使用方法：**
1. 输入股票代码
2. 输入当前价格
3. 输入买入价格
4. 设置止损比例
5. 点击 **📊 检查** 按钮

**输出结果：**
- 当前状态（持有/止损/止盈）
- 盈亏比例
- 止损价和止盈价
- 操作建议

## 高级功能

### 仓位管理

**功能：** 根据风险控制原则计算合理的仓位大小

**核心公式：**
```
建议仓位 = 总资金 × 单笔风险比例 / 止损比例
```

**使用场景：**
- 新建仓时确定买入金额
- 控制单笔交易风险
- 遵循资金管理原则

**示例：**
- 总资金：100,000元
- 单笔风险比例：2%
- 止损比例：5%
- 建议仓位 = 100,000 × 0.02 / 0.05 = 40,000元

### 止损止盈检查

**功能：** 实时监控持仓，及时止损止盈

**止损价计算：**
```
止损价 = 买入价 × (1 - 止损比例)
```

**止盈价计算：**
```
止盈价 = 买入价 × (1 + 止盈比例)
```

**使用建议：**
- 每日收盘后检查一次
- 严格执行止损纪律
- 止盈可以分批进行

### 导出历史记录

**功能：** 将分析记录导出为 Excel 文件

**导出格式：**
- 文件格式：.xlsx
- 包含内容：时间戳、分析类型、股票代码、分析结果

**使用场景：**
- 复盘历史交易
- 统计分析效果
- 归档重要分析

## 常见问题

### Q1: 如何添加自选股？

**A:** 在左侧控制台输入股票代码，点击"⭐ 添加到自选"按钮即可。

### Q2: 数据更新频率是多少？

**A:** 实时数据每分钟更新一次，历史数据在每次分析时获取。

### Q3: 为什么有些功能显示"数据不足"？

**A:** 某些功能需要一定数量的历史数据（如30天以上），如果股票上市时间较短或数据不足，会显示此提示。

### Q4: 如何提高分析准确性？

**A:**
1. 使用多个指标综合分析
2. 结合市场情绪判断
3. 严格执行风险控制
4. 定期复盘和总结

### Q5: 支持哪些股票？

**A:** 支持所有 A 股（包括沪市、深市、北交所）。

### Q6: 如何联系技术支持？

**A:** 如有问题，请在 GitHub 提 Issue 或联系开发者。

### Q7: 数据来源是什么？

**A:** 数据来源于 AkShare 开源财经数据接口，数据实时可靠。

### Q8: 系统对电脑配置有什么要求？

**A:** 推荐配置：
- CPU：双核及以上
- 内存：4GB 及以上
- 网络：稳定的互联网连接

### Q9: 可以同时分析多只股票吗？

**A:** 可以使用"多股对比"功能，最多支持同时对比5只股票。

### Q10: 如何备份数据？

**A:** 历史记录会自动保存到 `data/` 目录，定期备份该目录即可。
