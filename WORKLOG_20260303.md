### 3.5 细粒度切片设计
```python
SLICES = [
    (120, 90),   # 远景背景
    (90,  60),   # 洗盘确认
    (60,  30),   # 洗盘执行  ← 净卖出=洗盘信号
    (30,  15),   # 方向切换  ← 净买入翻正=预警信号
    (15,   5),   # 起爆加速
    (5,    0),   # 封板冲刺  ← 净卖出=假突破信号
]
```

**模式分类：**
- `wash_and_launch`: T-60~30洗盘 + T-30~15翻正 → 可埋伏
- `fake_breakout`: T-5~0净卖出 → 假封板，次日高开低走
- `slow_rise`: T-15~5加速 → 有时间埋伏
- `fast_surge`: 急拉型 → 追进风险大
